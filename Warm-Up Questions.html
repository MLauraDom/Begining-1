What is HTML?

HTML is the HyperText Markup Language used to describe web pages. HTML is not a programming language; it is a markup language. HTML consists of a series of short codes typed into a text-file by the site author — these are the elements.

How many versions of HTML do you know?

There are 6 HTML Versions: HTML(1991); HTML 2.0(1995); HTML 3.2 (1997); HTML 4.01 (1999); XHTML (2000); HTML 5 (2014)

How does the basic anatomy of a web page look like?

Header, Navigation, Main Content(Articles), Sidebar, Footer

Could you describe the constituting parts of the Web page? Open your favorite website in your browser and try to find these parts in it.

Header Visible at the top of every page on the site. Contains information relevant to all subsequent web pages on a website (like site name or logo) and an easy-to-use navigation bar system. Hero image In web design, a hero image is a large web banner image. Main content The biggest region, containing content unique to the current page. Stuff on the side Information complementing the main content; information shared among a subset of pages; alternative navigation system; basically, everything not absolutely required by the page's main content. E.g. sidebar. Footer Visible at the bottom of every page on the site. Like the header, contains global information like legal notices or contact info.

What are HTML elements? What is the difference between opening and closing tags?

HTML consists of a series of short codes typed into a text-file by the site author — these are the elements. Tags are used to define where an HTML element starts and ends. A start (opening) tag is made up of an opening angle bracket (<) followed by the name of the element, one or more space separated attribute/value pairs, and for the end (closing) tag a closing angle bracket (>). The end tag consists of a forward slash (/) written before the element name.

Could you describe the Page Structure of an HTML document explaining each element?

!DOCTYPE html - This tag defines the document type and HTML version.

html - This tag encloses the complete HTML document and mainly contains a document header which is represented by head.../head and a document body which is represented by body.../body tags.

head - This tag represents the document's header which can keep other HTML tags like title, link etc.

title - The title tag is used inside the head tag to mention the document title. body This tag represents the document's body which keeps other HTML tags like h1, div, p etc.

What is an HTML block element?

A block-level HTML element always starts on a new line and takes up the full width available.

What is an HTML inline element?

An inline element doesn’t start on a new line and only takes up as much width as necessary.

What is the div tag and why is it important? What is the difference between div and span elements?

The div tag represents a container unit that can encapsulate other HTML elements and divide the document into sections or divisions. This element is very important and is primarily used to group other HTML elements. The difference between span and div is that div is a block level element and span is an inline element.

What is the purpose of semantic elements in HTML5?

Semantic elements define different parts of a web page.
